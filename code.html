                            //Get the string from the page
                            //controller function
                            
                            function getValue(){
                            
                                //make sure that alert is invisible
                                document.getElementById("alert").classList.add("invisible");
                            
                                //get user string from page
                                let userString = document.getElementById("userString").value;
                                
                                //check for palindrome
                                let returnObj = checkForPalindrome(userString);
                            
                                //display message to the screen
                                displayMessage(returnObj);
                            }
                            
                            //check for a palindrome
                            function checkForPalindrome(userString){
                            
                                //convert to Lowercase
                                userString = userString.toLowerCase();
                            
                                //remove spaces and special characters
                                let regex = /[^a-z0-9]/gi;
                                userString = userString.replace(regex,"");
                            
                                let revString = [];
                                let returnObj = {};
                            
                                for (let index = userString.length - 1; index >= 0; index--) {
                                    revString += userString[index];      
                                }
                            
                                if (revString == userString) {
                                    returnObj.msg = "Well done! You've entered a palindrome!"
                                }
                                else{
                                    returnObj.msg = "Sorry! You did not enter a palindrome."
                                }
                            
                                returnObj.reversed = revString;
                            
                                return returnObj;
                            }
                            
                            function displayMessage(returnObj){
                            
                                document.getElementById("alertHeader").innerHTML = returnObj.msg;
                                document.getElementById("msg").innerHTML = `Your phrase reversed is: ${returnObj.reversed}`;
                                document.getElementById("alert").classList.remove("invisible");
                            
                            }

The Code Is Structured In Three Functions...

getValue

First, the alert message is set to "invisible", so that it doesn't show on start up. The user string is then obtained from the page using DOM. The userstring is checked for a palindrome by calling the function checkForPalindrome, then after this, the message is displayed to the user by calling the function displayMessage.

checkForPalindrome

This function checks whether the user has entered a palindrome. First, the user string is converted to lowercase. Then in order to remove any special characters, a regex statement is used. We ask the program to match the characters in the string to a-z and 0-9 - anything else is removed, including the space between characters. Since a reversed string is needed, revString is declared as an array. The user string is then reversed using a For Loop. Before checking for a palindrome, returnObj must be declared as an object to store various properties. After the For Loop, the palindrome is checked using if/else statements. In other words if the revString is equal to userString, then a message (...) will be displayed. Otherwise, then a message (...) will be diplayed. The reversed string is stored in the returnObj object, and the object is returned to the page.

displayMessage

This function display messages to the screen, from the user input.
